你的运行环境缺少 GraphicsMagick，因此本软件不能正常工作。
要安装 GraphicsMagick，你可以遵循以下步骤：

Windows:

到以下链接下载 GraphicsMagick 安装包：
点击去下载
完成安装后重新启动本软件即可，你有可能需要重启电脑以完成环境变量的刷新。

MacOSX:

在终端执行以下命令以安装 GraphicsMagick：
$ brew install graphicsmagick

Linux:

在不同发行版的 Linux 下，拥有不同的安装方式，在此仅列举最常用的发行版安装：
    Ubuntu: $ sudo apt install graphicsmagick
    CentOS: $ sudo yum install GraphicsMagick
